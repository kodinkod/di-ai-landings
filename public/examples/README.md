# Примеры изображений для лендинга

## Куда вставлять изображения

Поместите ваши изображения в эту папку (`/public/examples/`)

## Необходимые изображения

### Нана Банана примеры:

1. **styling-example.jpg** - Пример стилизации фото
   - Желательно показать До/После
   - Рекомендуемый размер: 800x600px

2. **weather-example.jpg** - Пример замены погоды
   - Фото с измененной погодой (солнце → дождь или наоборот)
   - Рекомендуемый размер: 800x600px

3. **id-photo-example.jpg** - Пример фото на документы
   - Портрет на документы
   - Рекомендуемый размер: 600x800px (вертикальное)

4. **retouch-example.jpg** - Пример ретуши
   - До/После ретуши портрета
   - Рекомендуемый размер: 800x600px

5. **outfit-example.jpg** - Пример примерки одежды
   - Фото с замененной одеждой
   - Рекомендуемый размер: 600x800px (вертикальное)

### OCR пример:

6. **ocr-example.jpg** - Пример изображения с текстом
   - Фото документа, чека или рукописного текста
   - Текст должен быть четким и читаемым
   - Рекомендуемый размер: 800x600px

## Как добавить изображения

1. Поместите файлы в эту папку
2. Убедитесь что имена файлов совпадают с указанными выше
3. В компонентах уже настроены пути к изображениям

## Где раскомментировать код

### В NanaBananaShowcase.jsx (строка ~60):
Раскомментируйте:
\`\`\`jsx
<img
  src={\`/examples/\${item.imagePlaceholder}\`}
  alt={item.title}
  className="w-full h-full object-cover group-hover:scale-110 transition-transform duration-500"
/>
\`\`\`

### В OCRShowcase.jsx (строка ~35):
Раскомментируйте:
\`\`\`jsx
<img
  src="/examples/ocr-example.jpg"
  alt="OCR Example"
  className="w-full h-full object-cover"
/>
\`\`\`

И в строке ~55 добавьте пример распознанного текста:
\`\`\`jsx
<p>
  Ваш пример распознанного текста<br />
  из изображения ocr-example.jpg
</p>
\`\`\`

## Оптимизация изображений

Для лучшей производительности:
- Используйте формат WebP или оптимизированный JPEG
- Рекомендуемое качество: 80-85%
- Максимальный размер файла: 200-300 KB

## Инструменты для оптимизации

- https://squoosh.app/ - онлайн оптимизация
- https://tinypng.com/ - сжатие PNG/JPEG
- ImageOptim (macOS) - десктоп приложение
