# Быстрый старт

## За 5 минут до запуска

### Шаг 1: Замените URL бота (2 минуты)

Найдите и замените `YOUR_BOT_USERNAME` на имя вашего бота:

\`\`\`bash
# macOS/Linux
grep -r "YOUR_BOT_USERNAME" src/

# Замените вручную в:
# - src/components/Hero.jsx
# - src/components/UseCases.jsx
# - src/components/CTA.jsx
# - src/components/Footer.jsx
\`\`\`

### Шаг 2: Добавьте логотип (1 минута)

1. Положите ваше изображение в папку `public/` (например, `public/logo.png`)
2. Откройте `src/components/Hero.jsx`
3. Найдите комментарий `{/* ВСТАВЬТЕ СЮДА ВАШ ЛОГОТИП */}`
4. Замените блок на:

\`\`\`jsx
<img
  src="/logo.png"
  alt="AI Assistant"
  className="w-full h-full object-contain"
/>
\`\`\`

### Шаг 3: Запустите проект (1 минута)

\`\`\`bash
npm run dev
\`\`\`

Откройте http://localhost:5173

### Шаг 4: Проверьте (1 минута)

- [ ] Кликните на кнопку "Открыть в Telegram" - должна открыться ссылка на ваш бот
- [ ] Прокрутите страницу - все секции должны отображаться
- [ ] Измените размер окна - проверьте адаптивность

## Готово к продакшену?

\`\`\`bash
npm run build
\`\`\`

Папка `dist/` готова к деплою на любой хостинг.

## Что дальше?

Смотрите `CUSTOMIZATION.md` для детальной настройки:
- Добавление примеров изображений
- Изменение кейсов использования
- Настройка цветов
- Добавление новых секций

## Проблемы?

1. **Не устанавливаются зависимости**: Проверьте версию Node.js (нужна 16+)
2. **Не запускается dev-сервер**: Убедитесь что порт 5173 свободен
3. **Не работают ссылки на Telegram**: Проверьте что заменили все `YOUR_BOT_USERNAME`
